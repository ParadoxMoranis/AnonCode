# Ablation Runners

This directory contains lightweight ablation runners for the anonymous code release.

## Files

- `run_isolength_ablation.py`
  - Runs iso-length translation with iterative reflow.
- `run_no_isolength_ablation.py`
  - Runs translation without iso-length control as a comparison setting.

## Expected Data

These scripts expect external experiment data prepared outside this repository.

## Output

Recommended output directories should also be created outside the repository or ignored by git.
